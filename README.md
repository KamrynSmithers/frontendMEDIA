## Features
IMMEDIATELY - This is going to be made into a social media kind of review site. Will be big with the right marketing and investments! 
Review books, movies, and music, in one place. Or, just explore ;)

## Site already has: 
- Playable movies (including new)
- Metadata on books, movies, and music
- Landing page w carousel of media underneath 
- Login/Register page
- Media Choices page
- Review sections/pages with comment sections at the bottom of everything you click on
